# Classificação: Aprendendo a Classificar Dados com Machine Learning 📊🤖

Este repositório contém o projeto prático e os estudos desenvolvidos durante o curso de **Classificação com Machine Learning** da Alura. O objetivo principal do projeto foi construir, avaliar e otimizar modelos preditivos para resolver problemas de classificação de dados (como a previsão de churn ou campanhas de marketing), utilizando a biblioteca **Scikit-Learn**.

---

## 🎯 Objetivo do Projeto

O projeto consiste na análise de uma base de dados de clientes para prever comportamentos de engajamento (classificação binária). Através do pipeline de Ciência de Dados, exploramos os dados, tratamos inconsistências, treinamos diferentes algoritmos de Machine Learning e avaliamos suas métricas de performance para encontrar o modelo ideal para o negócio.

## 🧠 Conceitos e Algoritmos Estudados

Durante o desenvolvimento do projeto, foram implementados e comparados os seguintes algoritmos de classificação:

* **K-Nearest Neighbors (KNN):** Classificação baseada em distância e similaridade entre os dados.
* **Bernoulli Naive Bayes:** Modelo probabilístico ideal para recursos que possuem valores binários.
* **Decision Tree (Árvore de Decisão):** Modelo que cria uma estrutura de árvore de decisão baseada em regras de quebra dos dados.

---

## 📈 Métricas de Avaliação Implementadas

Para garantir que o modelo escolhido fosse o mais eficiente, estruturamos a avaliação além da acurácia simples, analisando profundamente a **Matriz de Confusão**:

* **Acurácia (Accuracy):** Proporção geral de acertos do modelo.
* **Precisão (Precision):** Proporção de acertos entre as classificações positivas feitas pelo modelo.
* **Recall (Sensibilidade):** Capacidade do modelo de encontrar todos os casos positivos reais.
* **F1-Score:** Média harmônica entre precisão e recall, ideal para cenários com dados desbalanceados.
